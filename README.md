# plistener

<!-- summary -->

watch OSX plist preference files and record changes.

<!-- /summary -->
<!-- description -->

this is alpha-as-fuck. please use with caution.

it watches preference files stored as `.plist` on OSX and reports changes.

<!-- /description -->
